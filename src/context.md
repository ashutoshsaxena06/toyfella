Prompt for Co-Pilot
Task: Develop Customer-Facing Frontend for E-Commerce Website

Requirements:

Shop Page:

Display a grid of products with:
Product name
Price
Thumbnail image
Short description
"Add to Cart" button
Implement filters and sorting options (e.g., by category, price, and popularity).
Product Detail Page:

Provide detailed information about the product, including:
Larger product images (with a gallery)
Full description
Price and available stock
"Add to Cart" and "Buy Now" buttons
Customer reviews and ratings
Shopping Cart Functionality:

Create a cart component that:
Displays selected items with quantities, prices, and total cost
Allows users to update quantities or remove items
Provides a "Proceed to Checkout" button
User Authentication:

Design login and sign-up pages with:
Input fields for email and password
Option for social login (e.g., Google, Facebook)
Error handling for incorrect login attempts
Order History Page:

Allow users to view their past orders, including:
Order summary (products, prices, and order status)
Ability to reorder items from past orders
Support Page:

Include a FAQ section and a contact form for customer inquiries.
Ensure easy navigation to find help or support.
About Page:

Provide information about the company, mission, and values.
Include team bios or photos if applicable.
Responsive Design:

Ensure the entire frontend is mobile-friendly, adapting well to different screen sizes.
Test for usability on various devices.
Aesthetic and UX Design:

Implement a clean, modern design with:
Intuitive navigation
Clear call-to-action buttons
Consistent color scheme and typography
Use high-quality images and engaging layouts to attract customers.
SEO Optimization:

Ensure all pages are optimized for search engines with proper metadata and keywords.
Additional Notes:

Follow best practices for accessibility to ensure all users can navigate the site.
Keep components modular and reusable for better maintainability.
Incorporate user feedback to enhance the design and functionality over time.
